# Stylist
A base approach to JavaScript

# Using in node
Install this and eslint (save-dev in new projects)
```
npm install @basedakp48/eslint-config --save-dev
npm install eslint --save-dev
```

Create `.eslintrc.js` in root directory
```
module.exports = {
    "extends": "@basedakp48",
};
```
Done! :D
